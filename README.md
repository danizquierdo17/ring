# 🌙 LUA Ring - Tu Compañero de Ciclo Menstrual

Ring es una aplicación móvil gratuita y de código abierto para el seguimiento del ciclo menstrual, diseñada con un enfoque absoluto en la **privacidad** y **seguridad** de tus datos personales.

## 🔒 Privacidad Primero

### Tus Datos Son Solo Tuyos
- **100% Local**: Todos tus datos se almacenan únicamente en tu dispositivo
- **Sin Servidores**: No enviamos ninguna información a servidores externos
- **Sin Cuentas**: No requiere registro ni inicio de sesión
- **Sin Rastreo**: Cero analytics, cero seguimiento, cero telemetría
- **Sin Publicidad**: Completamente libre de anuncios y monetización

### Seguridad de Datos
- Base de datos local encriptada
- Sistema de backup y restauración bajo tu control
- Exportación de datos en formato seguro
- Sin permisos innecesarios

## ✨ Características Principales

### 📅 Seguimiento Inteligente del Ciclo
- Registro fácil de inicio y fin del periodo
- Cálculo automático de fases del ciclo
- Predicción de próximos periodos
- Historial completo de ciclos

### 🔔 Notificaciones Seguras
- Recordatorios personalizables para:
  - Inicio esperado del periodo
  - Ventana fértil
  - Ovulación
  - Fin del periodo
- **Notificaciones locales**: Procesadas completamente en tu dispositivo
- Sin conexión a internet requerida
- Control total sobre qué notificaciones recibir

### 🌙 Vista Luna (Mandala)
- Visualización artística de tu ciclo
- Representación de fases lunares
- Ciclograma del periodo completamente editable
- Colores personalizables para cada fase
- Interfaz intuitiva y hermosa

### 📊 Calendario Completo
- Vista mensual de tu ciclo
- Marcadores visuales para cada fase
- Edición de eventos del ciclo
- Navegación fácil entre meses

### 📖 Prospectos Oficiales Incluidos

### 🎨 Personalización
- **Temas**: Modo claro y oscuro
- **Idiomas**: Español e Inglés
- **Colores**: Personaliza los colores de cada fase del ciclo
- **Notificaciones**: Configura exactamente qué recordatorios quieres

### 💾 Backup y Restauración
- Exporta todos tus datos de forma segura
- Restaura desde backup cuando cambies de dispositivo
- Formato de datos legible y portable

## 🚀 Tecnología

Ring está construido con tecnologías modernas y confiables:

- **React Native + Expo**: Aplicación nativa multiplataforma

## 🛡️ Compromiso con la Privacidad

En un mundo donde tus datos de salud son valiosos para las empresas, Ring toma una postura diferente:

1. **Sin Monetización de Datos**: Nunca venderemos, compartiremos o monetizaremos tus datos
2. **Sin Terceros**: No integramos SDKs de terceros que puedan acceder a tu información
3. **Transparencia Total**: El código es abierto para que puedas verificar nuestras promesas
4. **Control Total**: Tú decides qué hacer con tus datos, cuándo exportarlos y cuándo eliminarlos
